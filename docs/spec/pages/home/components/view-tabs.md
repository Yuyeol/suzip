# View Tabs

## 레이아웃

```
┌────────────────────────────────────────────┐
│ [전체보기]  [폴더보기]                      │
└────────────────────────────────────────────┘
```

## 기능

**전체보기 탭**
- 모든 링크 카드 표시
- 검색바 + 정렬바 활성화
- 폴더 필터 가능

**폴더보기 탭**
- 폴더 카드 그리드 (2-column)
- 검색바 비활성화 (또는 숨김)
- 정렬바는 공유 (폴더명 정렬)

## 특징

- 탭 전환 시 URL state 변경 (`?view=all` / `?view=folders`)
- 선택된 탭은 primary 색상 + 밑줄
- 비선택 탭은 muted 색상
- width: 100%

## 상태 관리

- URL 파라미터로 상태 저장
- 새로고침 시에도 선택 유지
- 공유 시 같은 뷰로 열림
