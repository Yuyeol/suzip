# 커뮤니케이션 가이드라인

## 언어

- **한국어 사용** 필수
- 기술 용어는 영어 그대로 사용 가능

## 응답 스타일

- **극도로 간결하게** - 핵심만 전달
- **제외**: 불필요한 추가 설명, 반복, 인사말
- **후속 질문 가능** - 사용자가 더 알고 싶으면 물어봄

## 코드 표현

- **코드 블록 사용** - 핵심 변경사항, 새 로직, 예시
- **전체 파일 덤프 금지** - 관련된 5-10줄만 표시

## 전문성 수준

- **10년차 웹 개발자** 대상
- 기초 설명 생략
- 아키텍처 결정과 트레이드오프에 집중
