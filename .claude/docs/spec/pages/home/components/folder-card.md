# Folder Card

## 레이아웃

```
┌──────────────────┬──────────────────┐
│ 폴더명      ⋮   │ 폴더명2     ⋮   │
│ 12개            │ 5개             │
└──────────────────┴──────────────────┘
```

## 구조

**상단**
- 폴더명 (title-4, normal)
- ⋮ 더보기 버튼 (드롭다운: 수정/삭제)

**하단**
- 아이템 개수 (body-3, light)

## 특징

- 2-column 그리드
- 클릭 시 해당 폴더의 링크 목록으로 필터링
- 더보기는 클릭 이벤트 전파 중지
- 카드 높이 고정 (일관성)

## 더보기 옵션

- 수정 (폴더명 변경)
- 삭제 (폴더 내 링크는 "폴더 없음"으로 이동)
