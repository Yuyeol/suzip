# Sort Selector

## 레이아웃

**드롭다운 버튼**
```
┌──────────────┐
│ 최신순    ▼  │
└──────────────┘
```

**모달 (클릭 시)**
```
┌────────────────────────────────────────────┐
│                                            │
│  ○ 최신순                                  │
│  ● 오래된순                                │
│  ○ 가나다순                                │
│                                            │
│  [닫기]                                    │
│                                            │
└────────────────────────────────────────────┘
```

## 기능

**드롭다운 버튼**
- 현재 선택된 정렬 방식 표시
- 클릭 시 모달 오픈

**모달**
- 3개 옵션: 최신순 / 오래된순 / 가나다순
- 라디오 버튼 스타일 (단일 선택)
- 선택 시 즉시 적용 + 모달 닫기
- 닫기 버튼 or 배경 클릭으로 닫기

## 정렬 로직

- **최신순**: `created_at DESC`
- **오래된순**: `created_at ASC`
- **가나다순**: `title ASC` (대소문자 구분 없음)

## 특징

- 모바일 앱 스타일 모달
- 선택 즉시 반영
- URL state로 정렬 상태 저장 (공유/북마크 가능)
