# Search Bar

## 레이아웃

```
┌────────────────────────────────────────────┐
│ 🔍  검색어 입력...                    [X]  │
└────────────────────────────────────────────┘
```

## 기능

**실시간 검색**
- 입력 후 500ms debounce
- API 호출하여 결과 필터링
- 빈 값이면 전체 목록 표시

**X 버튼**
- 검색어 삭제
- 전체 목록으로 복귀
