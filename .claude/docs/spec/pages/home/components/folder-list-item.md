# Folder List Item

## 레이아웃

```
┌─────────────────────────────────────────────────────┐
│ 폴더명  12개    ☆    수정 | 삭제                    │
└─────────────────────────────────────────────────────┘
```

## 구조

- **폴더명 + 개수**: 클릭 시 전체보기 탭으로 전환 + 해당 폴더 필터링
  - 폴더명: `text-base font-semibold text-foreground truncate`
  - 개수: `text-xs text-muted`
- **즐겨찾기 버튼**: `FavoriteButton` 컴포넌트 (entityType: "folder")
- **수정/삭제 버튼**: `FolderActions` 컴포넌트
  - 수정: `text-primary` 텍스트 버튼
  - 삭제: `text-danger` 텍스트 버튼 (삭제 중 disabled)
  - 구분선: 수정/삭제 사이 세로선

## 폴더 탭 전체 구조

```
┌─────────────────────────────────────────┐
│ [FolderForm - 생성 모드]                │  ← 수정 중이 아닐 때만 표시
├─────────────────────────────────────────┤
│ [FolderListItem 1]                      │
│ [FolderListItem 2]                      │
│ ...                                     │
└─────────────────────────────────────────┘
```

## 인라인 수정

- 수정 버튼 클릭 시 해당 항목이 `FolderForm` (edit 모드)으로 교체
- 수정 중에는 상단 생성 폼 숨김
- 완료/취소 시 원래 리스트 아이템으로 복원

## 삭제 동작

- 삭제 버튼 클릭 시 `confirm` 다이얼로그 표시
- 북마크 0개: "정말 삭제하시겠습니까?"
- 북마크 N개: "N개의 북마크가 저장되어 있습니다. 폴더를 삭제하시겠습니까?\n(북마크는 삭제되지 않고 '폴더 없음'으로 이동합니다.)"
- 삭제 중 버튼 disabled 처리

## 특징

- 1-column 리스트 (`space-y-2`)
- `router.replace`로 폴더 필터링 (뒤로가기 히스토리 누적 방지)
- `buildUrlWithParams` 유틸로 URL 생성
- 낙관적 삭제 (즉시 리스트에서 제거)
