# Link Card

## 레이아웃

```
┌────────────────────────────────────────────┐
│ 제목                               ☆   ⋮  │
│ URL 미리보기 한 줄...                      │
│ 2024.12.08 | example.com          [IMG]   │
│ #태그1 #태그2 (예정)                       │
└────────────────────────────────────────────┘
```

## 구조

**왼쪽 (메인 영역)**
- 제목 (title-4, normal)
- URL 미리보기/설명 한 줄 (body-2, light)
- 날짜 | 플랫폼 (body-3, light)
- 태그 (예정)

**오른쪽 상단**
- ☆ 별표 토글 (즐겨찾기)
- ⋮ 더보기 버튼 (드롭다운: 수정/삭제)

**오른쪽 하단**
- 썸네일 이미지

## 특징

- width: 100%
- 클릭 시 상세 페이지 이동
- 별표/더보기는 클릭 이벤트 전파 중지
