# Bookmark Detail Page

## 개요

북마크의 상세 정보를 표시하는 페이지

---

## 경로

`/bookmark/[id]`

---

## 페이지 구조

```
┌────────────────────────────────────────────┐
│ 북마크 제목                      ☆    ⋮    │
│ 폴더명 | 2024.12.08                        │
├────────────────────────────────────────────┤
│ URL                                        │
│ https://example.com              🔗        │
├────────────────────────────────────────────┤
│ 썸네일 (있을 때만)                          │
│ ┌──────────────────────────────────────┐   │
│ │          [이미지]                    │   │
│ └──────────────────────────────────────┘   │
├────────────────────────────────────────────┤
│ 설명 (있을 때만)                            │
│ 설명 텍스트...                              │
├────────────────────────────────────────────┤
│ 메모 (있을 때만)                            │
│ 메모 텍스트...                              │
└────────────────────────────────────────────┘
```

---

## 섹션별 상세

### 헤더 영역
- **제목**: `Text variant="title-2"`
- **즐겨찾기 버튼**: `FavoriteButton` (entityType: "bookmark")
- **더보기 버튼**: `MoreButton` (수정/삭제 드롭다운)
- **폴더명**: `folder?.name || "main"` (폴더 미선택 시 "main" 표시)
- **날짜**: `YYYY.MM.DD` 형식 (한국 로케일)
- 폴더명과 날짜는 `|`로 구분, 같은 줄에 배치

### URL 섹션
- 레이블: "URL" (`body-3, text-muted`)
- URL 링크: `text-primary`, 외부 링크 (`target="_blank"`)
- `ExternalLink` 아이콘 (16px)

### 썸네일 섹션 (조건부)
- `bookmark.thumbnail`이 존재할 때만 표시
- 레이블: "썸네일"
- `rounded-lg`, `h-48`, `object-cover`

### 설명 섹션 (조건부)
- `bookmark.description`이 존재할 때만 표시
- 레이블: "설명"

### 메모 섹션 (조건부)
- `bookmark.memo`가 존재할 때만 표시
- 레이블: "메모"

---

## 동작

### 삭제
- `MoreButton`의 삭제 옵션 선택
- `onDeleteSuccess`: `router.replace("/")` (홈으로 이동)

### 수정
- `MoreButton`의 수정 옵션 → `/bookmark/[id]/edit` 페이지로 이동

### 로딩/에러 상태
- 로딩 중: "로딩 중..." 텍스트 표시
- 북마크 없음: "북마크를 찾을 수 없습니다." 텍스트 표시

---

## 기술적 특징

- SSR 비활성화
- `useGetBookmark` + `useGetFolders` 훅으로 데이터 로드
- 폴더명은 folders 목록에서 `bookmark.folder_id`로 매칭
