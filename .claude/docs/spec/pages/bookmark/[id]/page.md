# Bookmark Detail Page

## 개요

북마크 상세 정보를 보는 페이지

---

## 페이지 구조

```
┌────────────────────────────────────────────┐
│ [Header]                                   │
├────────────────────────────────────────────┤
│                                            │
│  북마크 제목                    [⋮]         │
│  2024.12.28                                │
│                                            │
│  🔗 URL                                    │
│  https://example.com                       │
│                                            │
│  📝 설명                                   │
│  북마크 설명 텍스트...                      │
│                                            │
│  📁 폴더명                                 │
│                                            │
└────────────────────────────────────────────┘
```

---

## 컴포넌트 구성

**제목 영역**
- Text 컴포넌트 title-2
- 생성일: 제목 아래 작은 글씨 (body-3)
- 더보기: MoreVertical 아이콘 + Dropdown

**Dropdown 옵션**
- 수정: `/bookmark/[id]/edit` 이동
- 삭제: 삭제 확인 후 API 호출 → 홈 이동

**URL**
- Link 컴포넌트
- 새 탭에서 열기 (target="_blank")
- 텍스트 색상 primary

**설명**
- 없으면 표시 안 함

**폴더**
- 선택 안 했으면 "미분류"

---

## 사용자 플로우

1. 북마크 정보 조회
2. URL 클릭 → 새 탭에서 사이트 열기
3. 수정 → 수정 페이지 이동
4. 삭제 → 확인 모달 → 삭제 후 홈 이동
