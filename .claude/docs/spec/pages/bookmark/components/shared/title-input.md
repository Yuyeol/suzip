# Title Input

## 레이아웃

```
┌────────────────────────────────────────────┐
│ 제목                                        │
│ ┌────────────────────────────────────────┐ │
│ │ 제목을 입력하세요                       │ │
│ └────────────────────────────────────────┘ │
└────────────────────────────────────────────┘
```

## 기능

- 제목 입력 (선택)
- 비어있으면 URL 메타데이터에서 자동 추출된 값 사용
- 최대 100자

## 특징

- Input 코어 컴포넌트 사용
