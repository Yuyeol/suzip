# URL Input

## 레이아웃

```
┌────────────────────────────────────────────┐
│ URL *                                      │
│ ┌────────────────────────────────────────┐ │
│ │ https://example.com                    │ │
│ └────────────────────────────────────────┘ │
└────────────────────────────────────────────┘
```

## 기능

- URL 입력 (필수)
- 유효성 검사 (URL 형식)
- blur 시 메타데이터 자동 추출 (제목, 설명, 썸네일)
- 에러 표시 (잘못된 URL)

## 특징

- Input 코어 컴포넌트 사용
- debounce 500ms
