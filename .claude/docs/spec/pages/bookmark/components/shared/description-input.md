# Description Input

## 레이아웃

```
┌────────────────────────────────────────────┐
│ 설명                                        │
│ ┌────────────────────────────────────────┐ │
│ │ 설명을 입력하세요                       │ │
│ │                                        │ │
│ │                                        │ │
│ └────────────────────────────────────────┘ │
└────────────────────────────────────────────┘
```

## 기능

- 설명 입력 (선택)
- 비어있으면 URL 메타데이터에서 자동 추출된 값 사용
- 최대 500자
- 여러 줄 입력 가능

## 특징

- textarea 사용
- 3줄 높이
