# Folder Selector

## 레이아웃

```
┌────────────────────────────────────────────┐
│ 폴더                                        │
│ ┌────────────────────────────────────────┐ │
│ │ 📁 개발 자료                        ▼  │ │
│ └────────────────────────────────────────┘ │
└────────────────────────────────────────────┘
```

**모달 (클릭 시)**
```
┌────────────────────────────────────────────┐
│                                            │
│  ○ 폴더 없음                                │
│  ● 개발 자료                                │
│  ○ 디자인 레퍼런스                          │
│  ○ 읽을거리                                 │
│                                            │
│  [닫기]                                     │
│                                            │
└────────────────────────────────────────────┘
```

## 기능

- 폴더 선택 드롭다운 (선택)
- 기본값: "폴더 없음"
- 클릭 시 모달 오픈
- 라디오 버튼 선택

## 특징

- 홈의 SortSelector와 동일한 모달 패턴
- 폴더가 없을 경우에도 사용 가능
- React Hook Form Controller 내장
