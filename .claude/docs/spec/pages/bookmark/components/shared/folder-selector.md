# Folder Selector

## 개요

북마크 폼에서 폴더를 선택하는 드롭다운 컴포넌트

## 레이아웃

```
┌────────────────────────────────────────────┐
│ 폴더                                       │
│ ┌──────────────────────────────────────┐   │
│ │ 📁 선택된 폴더명              ▼     │   │
│ └──────────────────────────────────────┘   │
│ ┌──────────────────────────────────────┐   │
│ │ 폴더 없음                            │   │
│ │ 📁 폴더1                             │   │
│ │ 📁 폴더2                             │   │
│ └──────────────────────────────────────┘   │
└────────────────────────────────────────────┘
```

## Props

```typescript
interface Props<T extends FieldValues> {
  name: Path<T>;
  control: Control<T>;
}
```

## 기능

- **필수** 필드 (`rules.required: "폴더를 선택해주세요"`)
- `DropdownSelect` 컴포넌트 사용 (드롭다운 방식)
- React Hook Form `Controller` 통합
- 옵션 목록:
  - `{ value: undefined, label: "폴더 없음" }` (기본 옵션)
  - 동적 폴더 목록 (`useGetFolders` 훅)
- 폴더 선택 시: `📁 폴더명` 형식으로 표시
- 미선택 시: "폴더 선택" placeholder
- 로딩 중: "로딩 중..." placeholder

## 특징

- `fullWidth={true}` - 전체 너비
- `min-w-[280px]` - 드롭다운 콘텐츠 최소 너비
- 에러 시 에러 스타일 표시 + 에러 메시지
- 제네릭 타입으로 다양한 폼에서 재사용 가능
