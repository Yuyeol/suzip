# Folder List

## 레이아웃

**일반 모드**
```
┌────────────────────────────────────────┐
│ 📁 개발 자료 (12)    [수정] [삭제]      │
└────────────────────────────────────────┘
┌────────────────────────────────────────┐
│ 📁 디자인 (5)        [수정] [삭제]      │
└────────────────────────────────────────┘
```

**수정 모드** (인라인)
```
┌────────────────────────────────────────┐
│ [개발 자료 입력...] [수정 완료] [취소]  │
└────────────────────────────────────────┘
```

## Props

없음 (자체 데이터 조회 및 상태 관리)

## 기능

- 폴더 목록 자동 조회 및 렌더링
- 각 아이템: 폴더 정보 + 수정/삭제 버튼
- 수정 클릭 시 해당 아이템만 Input으로 변환 (인라인 수정)
- 삭제 기능
- 로딩 상태 처리
- 빈 상태 처리

## 내부 구현

- `useGetFolders`: 폴더 목록 조회
- `usePatchFolder`: 폴더 이름 수정
- `useDeleteFolder`: 폴더 삭제
- `editingId`, `editValue` state로 인라인 수정 관리
